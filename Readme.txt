Readme:

This app aims to bring create a community of gardeners. 

How to use?

- Index.html is the starting file for the front end. And contains the start ui controls for the application.
- Share a plant.html contains the base screens for the connecting pages.
- The front end uses javascript and HTML controls.

- WE have an asp.net REST Api for connecting this application with the backend application. 
- The rest api has a visual studio 2015 project and needs an ISS server to execute.
- We only have a couple of tables in localDB with the project that are connected via entity framework.
